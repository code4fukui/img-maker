# img-maker
DALL-E OpenAIを使ったイメージ生成ツールです。

## 機能
- テキストから画像を生成できます
- 生成した画像はキャッシュに保存されるため、2回目以降の生成時は高速に表示されます

## 必要環境
- Deno
- OpenAIのAPIキー

## 使い方
1. リポジトリをクローンします
2. `.env`ファイルにOpenAIのAPIキーを設定します
3. `deno serve -A --port 8888 img-maker.js`コマンドを実行してサーバーを起動します
4. ブラウザで`http://localhost:8888/`にアクセスします
